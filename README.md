# Personal-finance-guide-

React Personal Finance Guide app

A simple **Personal Finance Guide web application** built with **React, Vite, and Tailwind CSS**.  
This project was created as part of a course assignment to demonstrate modern frontend development and basic financial tracking features.

The app allows users to:
- Track monthly income
- Add and view expenses
- Calculate savings automatically
- Monitor savings progress
- Learn basic personal finance concepts

---

# Project Preview

The application contains three main sections:

### Dashboard
Displays:
- Income
- Total expenses
- Savings
- Savings rate
- Savings progress bar

### Budget Tracker
Allows users to:
- Enter monthly income
- Add expenses
- View a list of expenses
- Automatically calculate savings

### Finance Education
Provides short guides about:
- 50/30/20 budgeting rule
- Emergency funds
- Importance of early investing

# Tech Stack

This project uses the following technologies:

- **React** – UI library
- **Vite** – Fast development environment
- **Tailwind CSS** – Utility-first styling
- **Framer Motion** – UI animations
- **Lucide React** – Icons
- **shadcn/ui** – UI components
